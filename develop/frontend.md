# 前端开发

> 目前前端页面项目是基于`Genshin-Impact-Wish-Simulator`的二次开发项目，使用`svelte`语言开发

## 安装依赖

```bash
npm install
```

## 调试

源码在`src`文件夹下，可通过以下命令来对页面进行实时调试

```bash
npm run dev
```

## 构建
### 页面构建
可通过以下指令进行页面构建，构建后的文件将在`.vercel/output/static`目录下

```bash
npm run build
```
